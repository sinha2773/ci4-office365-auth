
	1. run "git clone https://github.com/sinha2773/ci4-office365-auth.git" on your htdocs/www folder
	2. run "cd ci4-office365-auth"
	3. run "composer update"
	3. create database msg_auth and import msg_auth.sql or you can config yourself
	4. browse http://localhost/ci4-office365-auth/public/
	5. you can update necessary configurations at public/assets/js/msauth.js
